# Charm Loadouts

A simple and lightweight mod for **Hollow Knight** that allows you to save and switch between 3 different charm builds instantly while resting at a bench.

## Features
* **3 Save Slots**: Save your favorite charm combinations.
* **Easy Controls**: Use hotkeys to save and load.
* **Bench Restricted**: Loading only works at a bench to keep the gameplay balanced.
* **In-Game Menu**: Check which slots are currently filled via the Mod Menu.

---

## How to Use

### Controls
* **Save Build**: Press `Left Shift` + `1`, `2`, or `3` to save your currently equipped charms to that slot.
* **Load Build**: Press `1`, `2`, or `3` while sitting at a bench to instantly equip that build.

> **Note**: You must have the charms in your inventory to load them. If you haven't found a charm yet, it will simply be skipped.

### Mod Menu
You can find the **Charm Loadouts** section in the "Mods" menu. It will show the status of your slots:
* **есть**: Build is saved in this slot.
* **нету**: Slot is empty.

## Installation
1. Install [Lumafly](https://github.com/Pr0metheus99/Lumafly) or [Scarab](https://github.com/fifty-nine/Scarab).
2. Search for **Charm Loadouts** in the mod list.
3. Click Install.
